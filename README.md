# WaffleWare

NOTE: do not press quit. This project is built for the embedded Godot project runner so pressing quit will freeze the game. 

A chaotic, fun-packed minigame collection! 

# Features:

- Minigame 1: A mini platformer where you collect apples. 
- Minigame 2: A clicker game where you click waffles
- Minigame 3: A reaction game where you catch a coconut
- Settings: Adjustable music and SFX volume.
